# SkyrimNet_SexLab

Adds SkyrimNet support to SexLab.

# Animation Stage descriptions 

`SkyrimNet_SexLab\animations\local\file.json
~~~
{
    id:String 
    name:String,
    stages:[
        {
            stage:Int,
            descriptions:[
                {
                    source:String
                    version:middle|inja
                    description:String 
                }
            ]
        }
    ]
}
~~~
